Firefoxos-ecobattery

Aplicación para dispositivos Firefox OS, usando las API de Batería y de Vibrador

Download this project as a .zip file Download this project as a tar.gz file

EcoBattery

Esta es una aplicación sencilla que ayuda a cuidar el medio ambiente, ¿como? pues ahorrando energía eléctrica. Por una parte es una simple interface que muestra los detalles de la batería de nuestro dispositivo, principalmente el porcentaje de carga que tiene, además de que es capaz de detectar si el dispositivo está o no conectado a una fuente de alimentación, de ser así se activará un evento que comprobará si la batería esta completamente llena, osea al 100%, en ese momento se reproducirá un sonido y el telefono vibrará durante 1000ms para avisarle al usuario que ya tiene que ser desconectado, de está manera se evita el uso extendido de energía eléctrica.

Tecnologías Usadas

Para hacer posible esta aplicación usé diversas tecnologías web, como son:
♦ HTML5
♦ CSS3
♦ JavaScript
♦ JQuery
♦ Firefox OS - WebApi
